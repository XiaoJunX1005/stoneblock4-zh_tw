## ClassicPipes（經典管線）

### 核心概念
- Pipe：管線
- Item Pipe：物品管線
- Fluid Pipe：流體管線
- Networked Pipe：網路管線
- Network：網路
- Route / Routing：路由 / 路由（建議固定用「路由」）
- Default Route：預設路由
- Destination：目的地
- Provider：供應（供應管線）
- Request：請求（請求管線 / 請求數量）
- Stocking：補給（補給管線 / 主動補給）
- Matching：匹配（匹配管線）
- Recipe：配方（配方管線 / 設定配方）
- Storage：儲存（儲存管線）

### 行為與操作
- Extract：抽取
- Insert：輸入
- Eject：彈出 / 掉出（物品從管線掉出）
- Delete / Destroy：刪除 / 銷毀（物品已銷毀）
- Deploy：部署（像發射器一樣部署物品）
- Connect / Connected：連接 / 已連接
- Disconnect：中斷連接 / 斷開
- Adjust：調整（管線已調整）
- Request Amount：請求數量
- Cancel：取消
- Timed out：逾時

- Filter：過濾器
- Tag：標籤（Tag）
- Mod：模組（Mod）
- Tag Filter：標籤過濾器
- Mod Filter：模組過濾器
- Match Components：匹配組件（耐久度、附魔等）
- Components：組件（耐久度、附魔等）
- Leave One：保留一個

### 模式與設定
- Active Stocking：主動補給
- Blocking Mode：一次一個（阻塞模式）
- Craftable：可合成
- Sort By：排序方式
- Ascending：低到高
- Descending：高到低
- A to Z：A 到 Z
- Z to A：Z 到 A
- Page：第 %s/%s 頁

### 容器與加工
- Container：容器

- Processing System：加工系統
- Adjacent：相鄰
- Junction / Intersection：交叉口
